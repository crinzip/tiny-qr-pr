# Tiny QR

Tiny and quick way of creating QR codes straight in Premiere.

Made for Mac, but tested and functional on Windows (hopefully, fingers crossed emoji)

Built on UXP, so only supports Premiere 25.6 and above

## Download

Grab the latest `.ccx` file from the [**Releases**](https://github.com/crinzip/tiny-qr-pr/releases) tab.

If there's an update, you'll see a popup notification in the plugin. Clicking it will take you to the latest release, where you can download the updated `.ccx` file and install it over your current version.
